# 2048 Galaxy

2048 Galaxy is a simple tile colliding game like the traditional 1024 and 2048 game that came in old keypad mobile phones.
with that reference i made this one in a space theme.
it has simple animations for swapping, removing and undo options with sounds.

## how to play

- move with arrow keys or wasd
- on phone just swipe
- same numbers touch = merge
- every move spawns new tile
- game over when board full and no merge left

## extra buttons

- undo: go one step back
- swap: pick 2 tiles and swap them
- remove: delete one tile
- new game: reset all
- sound button: mute/unmute

## goal

make 2048 tile.
then if u want keep going for big score.
